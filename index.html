September 2024

RESPONSIBILITIES

Fullstack Developer

TECH STACK

SHORT DESCRIPTION

The system provides a store management platform that allows users to view available products. Notaris App

August 2024

RESPONSIBILITIES

Fullstack Developer

TECH STACK

Flutter Laravel MySQL

SHORT DESCRIPTION

Applications that are used to manage file data and file archives in notary offices can be managed dynamically by each user. Designed as an Android application, it provides a mobile-friendly interface that allows notary staff to access and organize files on the go. The app supports easy document retrieval, categorization, and secure storage directly from their mobile devices. With push notifications and offline access, users can stay updated on file statuses and continue their work even without an internet connection, enhancing productivity and accessibility in any setting. To streamline the backend operations, the application utilizes Laravel as an API, ensuring efficient data handling and seamless integration with existing systems.
